Note App

View Source Code on GitHub

The NoteApp is a user-friendly web application built using the MERN stack, combining React with Vite for the frontend and Node.js, Express.js, and MongoDB for the backend. This project offers secure user authentication with signup and login pages, ensuring only registered users can access the app. Users can create, edit, delete, and search notes efficiently, with toast notifications enhancing the user experience. The app's intuitive design, styled using Tailwind CSS, makes note management seamless, while the logout feature ensures session security. Developed as a self-study project, it showcases the integration of CRUD operations and modern web technologies. This foundational project highlights key MERN stack functionalities, laying the groundwork for more advanced full-stack development projects in the future.
