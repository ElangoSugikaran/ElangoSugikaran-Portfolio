Sugikaran Store: A Laravel-Based E-Commerce Platform



View Source Code on GitHub

The Sugikaran Store E-commerce Web Application is built using the Laravel 11 framework, offering a seamless shopping experience for customers. Users can easily register, verify their email, and reset passwords through secure email links. The system supports product browsing, adding/removing items from the cart, and secure payments via Stripe or Cash on Delivery. Customers can also track order and payment statuses from their personal account. For admins, the platform provides robust CRUD operations, allowing them to manage categories, products, and orders. Admins can update delivery statuses, view order details, and print invoices. The system includes multi-authorization features, ensuring secure access and efficient management of store operations.

Skills Acquired

Through this project, I developed and honed my skills in Laravel 11, working with various advanced functionalities to successfully build the E-commerce web application. The project involved utilizing a robust database architecture 💾, implementing multi-authentication 🔐, and integrating a custom template 🖼️ for a smooth user experience.

Key features included building complete CRUD operations 🔄, sending user-friendly toastr messages 🍞 and sweet alerts 🍬 for notifications, ensuring proper pagination 📄, handling image uploading and viewing 📸, implementing an efficient search function 🔍, redirecting routes ➡️, and managing foreign keys 🔗 in the database. Additionally, I integrated PDF printing 🖨️, established secure mail connections ✉️, handled email verification ✅, and incorporated online payment processing 💳 through Stripe.

Working on this project not only solidified my understanding of Laravel but also expanded my knowledge of e-commerce systems, making this a key milestone in my development journey.
